# GDDXX-Jarvis 迭代记录：2026-07-21 / Round 03

时间：19:08-19:16（Asia/Shanghai）

本轮原则：长对话可回看、流式过程可感知、每条回复可操作。

## 已发现并解决的问题（10/10）

1. **新消息会强制滚到底部，打断用户回看。** 仅在用户位于底部附近时自动跟随。
2. **停止跟随后没有返回最新消息的快捷入口。** 增加“最新消息”浮动按钮。
3. **超过 40 条的历史被静默截断。** 显示较早消息数量，并支持展开/收起全部记录。
4. **流式回复与完成回复视觉相同。** LIVE 消息增加低干扰闪烁光标。
5. **消息来源渠道不可见。** 每条消息显示 TUI、语音、LIVE 等渠道。
6. **角色只显示英文。** 改成“您 / YOU”“贾维斯 / JARVIS”“系统 / SYSTEM”。
7. **不能逐条复制回复。** Jarvis 消息提供复制操作与短暂“已复制”反馈。
8. **只能重播最后一条回复。** 每条 Jarvis 消息都提供独立重播操作。
9. **长回复占满整个历史栏。** 超过阈值自动折叠，并可展开或收起。
10. **等待回复时只有无限旋转图标。** 思考状态显示已等待秒数。
